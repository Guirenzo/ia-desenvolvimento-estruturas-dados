# IA para Desenvolvimento + Revisão de Estruturas de Dados I

Repositório acadêmico da equipe para a atividade **“IA para Desenvolvimento + Revisão de Estruturas de Dados I (Desafios de Code Interview)”**, com uso de **ChatGPT** como IA principal e implementações em **JavaScript**.

## Objetivo
Resolver **12 problemas** clássicos de estruturas de dados, com foco em:
- Arrays
- Linked Lists
- Stacks

Além da implementação, o projeto foi organizado para evidenciar:
- entendimento técnico dos problemas;
- justificativa da estrutura de dados escolhida;
- análise de complexidade;
- testes;
- documentação por problema;
- uso supervisionado de IA no processo.

## IA principal utilizada
**ChatGPT**

### Como a equipe utilizou a IA
A IA foi usada como apoio em:
- interpretação do enunciado;
- proposta de solução inicial;
- revisão da lógica;
- análise de complexidade;
- geração de casos de teste;
- padronização da documentação técnica.

As respostas sugeridas pela IA foram revisadas e refinadas pela equipe antes da versão final.

## Linguagem adotada
**JavaScript (Node.js)**

## Estrutura do repositório

```text
arrays/
linked-lists/
stacks/
docs/
shared/
```

- `arrays/`: problemas relacionados a vetores, busca, two pointers, prefix/suffix e busca binária.
- `linked-lists/`: problemas que exigem manipulação de ponteiros e reordenação de nós.
- `stacks/`: problemas com pilha tradicional, pilha com mínimo e monotonic stack.
- `docs/`: relatório final, roteiro de apresentação e modelo de documentação.
- `shared/`: utilitários auxiliares para listas encadeadas e testes.

## Como executar

### 1. Instalar dependências
Não há dependências externas obrigatórias. Basta ter o **Node.js** instalado.

### 2. Executar todos os testes
```bash
npm test
```

### 3. Executar um teste específico
Exemplo:
```bash
node arrays/two-sum/test.js
```
