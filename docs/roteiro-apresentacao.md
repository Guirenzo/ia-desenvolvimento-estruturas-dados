# Roteiro de apresentação

## 1. Abertura
Boa tarde. Vamos apresentar o trabalho **IA para Desenvolvimento + Revisão de Estruturas de Dados I**, no qual utilizamos inteligência artificial como apoio para resolver problemas clássicos de code interview relacionados a Arrays, Linked Lists e Stacks.

## 2. Objetivo
A atividade exigia a escolha de uma IA principal e a resolução de 12 problemas:
- 5 de Arrays
- 4 de Linked Lists
- 3 de Stacks

## 3. IA escolhida
A equipe escolheu o **ChatGPT** por seu apoio em:
- interpretação dos enunciados;
- geração de soluções iniciais;
- revisão;
- testes;
- documentação.

## 4. Metodologia
Fluxo adotado:
1. entender o problema;
2. identificar a estrutura central;
3. gerar uma solução inicial com apoio de IA;
4. revisar criticamente;
5. implementar em JavaScript;
6. testar;
7. documentar.

## 5. Destaques técnicos
### Two Sum
Mostra como um hash map reduz a solução de O(n²) para O(n).

### 3Sum
Combina ordenação, two pointers e tratamento de duplicados.

### Reorder List
Combina encontrar o meio, inverter metade da lista e intercalar os nós.

### Daily Temperatures
Aplica monotonic stack de forma eficiente.

## 6. Conclusão
O trabalho foi importante para revisar estruturas de dados e demonstrar o uso consciente da IA no processo de desenvolvimento, com foco em corretude, organização e documentação.
